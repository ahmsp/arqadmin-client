Bem-vindo ao

ArqAdmin - Sistema de Gestão dos Acervos do Arquivo Histórico de São Paulo.

O ArqAdmin é uma ferramenta para pesquisa e gerenciamento interno, dos acervos custodiados pela instituição.

Você está acessando o sistema como consulente, com acesso exclusivo para pesquisa. Os recursos de edição, restritos a usuários autorizados, não serão exibidos para o seu perfil.

Para dúvidas e sugestões acesse o menu de informações do sistema.

Boa pesquisa!
